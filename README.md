# E-commerce Sales Analysis

## Project Overview

This project analyzes e-commerce sales data from the TheLook Ecommerce dataset using Google BigQuery.

## Business Questions

1. What is the total revenue generated from completed orders?
2. Which order statuses occur most frequently?
3. How does revenue change over time?

## Tools

- Google BigQuery
- SQL
- GitHub

## Dataset

TheLook Ecommerce Dataset (BigQuery Public Dataset)
